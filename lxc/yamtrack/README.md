# yamtrack

[Yamtrack](https://github.com/FuzzyGrim/Yamtrack) is a YouTube Music and Spotify tracking application.

## 🛠️ Installation

This application was installed as a Proxmox LXC using the following configuration:

- **Node:** pve03
- **ID:** 125
- **OS:** Debian Trixie
- **URL:** [https://yamtrack.l.nicholaswilde.io](https://yamtrack.l.nicholaswilde.io)

## ⚙️ Config

The configuration is managed via a `.env` file in `/opt/yamtrack/src/.env`.

## 🚀 Upgrade

```bash
task update
```
